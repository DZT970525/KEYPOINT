# Magnetic tenting stand

Integrated MagSafe-compatible magnets are built into each keyboard half, allowing effortless attachment to MagSafe tenting stands and other compatible accessories. Users can quickly adjust the tenting angle to achieve a more ergonomic typing position, enhancing both comfort and productivity.
